Font Features

Themify Icons is a complete set of icons for use in web design and apps, consisting of 320+ pixel-perfect, hand-crafted icons that draw inspiration from Apple iOS 7. This icon set is a must have tool for web designers and developers.

One Font, 351 Icons

In a single collection, Themify Icons has 351 hand crafted icons.

CSS Control

Easily style icon color, size, shadow, and anything that's possible with CSS.

Infinite Scalability

Scalable vector graphics means every icon looks awesome at any size.

Desktop Friendly

To use on the desktop or for a complete set of vectors, check out the cheatsheet provided below.

Perfect on Retina Displays

Themify icons are vectors, which mean they're gorgeous on high-resolution displays.

Screen Reader Compatible

Themify Icons won't trip up screen readers, unlike other icon fonts.
