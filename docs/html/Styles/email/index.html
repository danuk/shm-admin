Email Templates

The Email Templates provided with Outline has it's styles in the head section. Do run the template through a CSS inliner before sending them out!

The included Email Tempalate styles are separate from the core Outline styles, thus require special mention. Due to the nature of how different email clients work with different emails, the email style has to be inline. We use the Ink CSS Framework to help craft emails that can be read at any devices - from smart phones to Outlook.

The Process

  • Design in Outlook
  • Add your own responsive styles in the head section of the document within the style tag.
  • Bring styles inline using a CSS Inliner like Inky's Inliner
  • Test to see if all your mails are are working, we recommend Litmus
  • Send out your mails!

For further details and documentation, please refer to Ink website.
